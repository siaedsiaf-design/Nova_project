1. فاحص المنافذ الأساسي (Network Port Scanner)

💡 **الشرح المفصل:** يقوم هذا السكربت بالاتصال المباشر بمجموعة من المنافذ (Ports) الافتراضية على خادم معين للتأكد مما إذا كانت مفتوحة أم مغلقة.

import socket
target = "127.0.0.1"
for port in range(1, 1025):
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.settimeout(0.1)
    if s.connect_ex((target, port)) == 0:
        print(f"[+] المنفذ {port} مفتوح ومتاح للاتصال! 🔓")
    s.close()

2. قناص معلومات الخادم (Banner Grabber)

💡 **الشرح المفصل:** يتصل بالمنفذ الخاص بالويب ويرسل طلباً بدائياً ليجبر الخادم على الرد بالسطر التعريفي الخاص به (Banner).

import socket
s = socket.socket()
s.settimeout(2)
try:
    s.connect(("127.0.0.1", 80))
    s.send(b"GET / HTTP/1.1\r\n\r\n")
    print(f"[!] بيانات السيرفر المكتشفة: {s.recv(1024).decode()}")
except:
    print("[-] تعذر جلب البانر.")

3. فاحص الروابط وحالة الويب (HTTP Status Checker)

💡 **الشرح المفصل:** يرسل طلبات HTTP تسمى GET ويقوم بتحليل كود الرد للتأكد من حالة اتصال الموقع الإلكتروني.

import requests
url = "https://www.google.com"
try:
    status = requests.get(url, timeout=3).status_code
    print(f"[✅] الرابط شغال تمام، كود الاستجابة: {status}")
except:
    print("[❌] الرابط مغلق أو أوفلاين.")

4. محاكي هجوم القوة الغاشمة (Brute Force Simulator)

💡 **الشرح المفصل:** تمر الدالة على قائمة مجهزة مسبقاً من كلمات المرور وتجربها واحدة تلو الأخرى حتى تجد الكلمة الصحيحة واختبار قوة الدفاع.

passwords = ["12345", "admin", "nova2026", "password"]
correct_one = "nova2026"
for attempt in passwords:
    if attempt == correct_one:
        print(f"[🎉] تم كسر التشفير بنجاح! كلمة السر هي: {attempt}")
        break

5. مستخرج عناوين الـ IP (DNS IP Resolver)

💡 **الشرح المفصل:** يعتمد على بروتوكول DNS لترجمة أسماء النطاقات المفهومة للبشر إلى عناوين رقمية (IP Addresses).

import socket
domain = "google.com"
ip = socket.gethostbyname(domain)
print(f"[🎯] عنوان الـ IP الحقيقي لموقع {domain} هو: {ip}")

6. مشفر النصوص السري (Simple Text Encrypter)

💡 **الشرح المفصل:** يقوم بعملية تشفير بدائية تعتمد على إزاحة القيم الحرفية بمقدار معين لحماية النصوص المخزنة.

text = "Nova Cyber"
encrypted = [ord(char) + 5 for char in text]
print(f"[🔒] النص المشفر كأرقام حماية: {encrypted}")

7. فاحص سلامة الملفات (File Hash Calculator)

💡 **الشرح المفصل:** يستعمل خوارزمية SHA-256 لتوليد بصمة فريدة وثابتة الطول لأي نص أو ملف لكشف التلاعب والتعديلات.

import hashlib
data = b"Nova Security Data"
file_hash = hashlib.sha256(data).hexdigest()
print(f"[🛡️] البصمة الرقمية الفريدة للملف: {file_hash}")

8. صائد ومستخرج الروابط الفرعية (Subdomain Finder Basic)

💡 **الشرح المفصل:** يقوم بتخمين واختبار النطاقات الفرعية التابعة للموقع الأصلي للبحث عن أي ثغرات أو خوادم مطورين منسية.

import requests
subs = ["admin", "blog", "mail", "dev"]
main_url = "google.com"
for sub in subs:
    url = f"https://{sub}.{main_url}"
    try:
        if requests.get(url, timeout=2).status_code == 200:
            print(f"[+] نطاق فرعي نشط مكتشف: {url}")
    except: pass

9. ساحب عناوين الـ Headers الأمنية (Security Headers Analyzer)

💡 **الشرح المفصل:** يفحص ترويسات الاستجابة القادمة من السيرفر للتأكد من تفعيل جدران حماية المتصفح الحركية.

import requests
res = requests.get("https://www.google.com")
headers = ["X-Frame-Options", "Content-Security-Policy"]
for h in headers:
    print(f"[📋] {h}: {res.headers.get(h, 'غير مفعّل ⚠️')}")

10. كاشف الأجهزة المتصلة بالشبكة (Local Ping Sweeper)

💡 **الشرح المفصل:** يرسل طلبات صدى حزم الـ ICMP بصورة تسلسلية إلى عناوين الأجهزة المتواجدة معك على نفس الشبكة المحلية.

import os
for i in range(1, 5):
    ip = f"192.168.1.{i}"
    response = os.system(f"ping -n 1 {ip}" if os.name == 'nt' else f"ping -c 1 {ip} > /dev/null")
    if response == 0:
        print(f"[💻] الجهاز ذو العنوان {ip} نشط متصل بالشبكة!")
